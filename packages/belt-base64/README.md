# @aegenet/belt-base64

> Base64

## 💾 Installation

```shell
yarn add @aegenet/belt-base64@^1.4.0
# or
npm i @aegenet/belt-base64@^1.4.0
```

## 📝 Usage

```typescript
import { toBase64, fromBase64 } from '@aegenet/belt-base64';

toBase64('Something'); // 'U29tZXRoaW5n'

fromBase64('U29tZXRoaW5n'); // 'Something'
```
